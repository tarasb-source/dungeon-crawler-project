# Dungeon Crawler

A top-down 2D dungeon crawler game built in **Java** with **JavaFX** for my COMP 301 final project.

## Demo Video

Watch the gameplay demo: https://youtu.be/MuQ_zDaxCNQ

## Overview

Dungeon Crawler is a top-down 2D game where the player navigates through a dungeon environment, interacts with enemies, and uses the game interface to track progress. It was built in Java with JavaFX. I worked on the project using an MVC structure to separate game logic, user input, and rendering. The project includes player movement, enemy interactions, scoring, UI screens, animations, and styled JavaFX views.

## Features

- Top-down player movement
- Enemy interactions
- Scoring system
- UI screens and scene transitions
- JavaFX animations
- Styled JavaFX views
- MVC-based project structure

## Technologies Used

- Java
- JavaFX
- CSS
- Object-Oriented Programming
- MVC Architecture
